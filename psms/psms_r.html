Математика и статистика (часть 2)

Работа в R

1. Установка R и RStudio

Для работы необходимо установить R и среду для удобной работы RStudio.
Cначала нужно установить R, затем RStudio.

Инструкция для Windows

  1. Для R: заходим сюда, кликаем Download R 3.6.3 for Windows, сохраняем файл .exe себе на компьютер, запускаем, следуем инструкциям.
  2. Для RStudio: заходим сюда, в All installers выбираем Windows, кликаем на файл с расширением .exe, сохраняем на компьютер, запускаем, следуем инструкциям. Рекомендуется не отказываться от добавления ярлыков.
  3. После установки либо на рабочий стол будет добавлен ярлык RStudio, либо в меню Пуск будет папка/ярлык RStudio. Если нет, в меню Пуск делаем поиск и вводим RStudio.

Инструкция для Mac OS

  1. Для R: заходим сюда, кликаем в Latest release на R-3.6.3.pkg, сохраняем файл .pkg себе на компьютер, запускаем, следуем инструкциям.
  2. Для RStudio: Заходим сюда, в All installers выбираем masOS, кликаем на файл с расширением .dmg, сохраняем на компьютер, запускаем, следуем инструкциям.
  3. После установки в Launchpad будет добавлен ярлык RStudio.
